# 📌 专利附图标记智能排版 (Markup Assistant)

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![Build](https://img.shields.io/badge/Build-Success-success.svg)

**专利附图标记智能排版** 是一款专为**专利代理师**和**知识产权工程师**量身定做的专业级浏览器端提效工具。旨在彻底消除您在撰写专利文件（实施例、专利要求、发明内容）时繁琐、极易出错的**附图标记（数字编号与部件名称）**双向倒置与校对工作。

只需输入一组标记定义，无论何种乱序排版，引擎均能帮您在六大场景文本之间实现一键无缝转换。

---

## ✨ 核心亮点特性

### 1. 🔀 六向全场景覆盖转换
涵盖了专利代理执业中涉及的所有主流部件名称撰写规范，支持以下六大流向转换：
- **发明内容 ⇒ 实施例** （自动给基础部件配上编号，并且**智能去除多余的“所述”字眼**）
- **发明内容 ⇒ 权利要求** （自动加上带括号的编号对，如 `机构（1）`）
- **实施例 ⇒ 发明内容** （剥离数字，还原纯净名称）
- **实施例 ⇒ 权利要求** （数字转括号格式）
- **权利要求 ⇒ 发明内容** （剥离括号编号）
- **权利要求 ⇒ 实施例** （括号编号变纯数字后置）

### 2. 🧠 极致容错的双向解析引擎
不要再纠结原文件里究竟写的是 “**1、机构A**”、“**2-机构B**”、“**机构C 3**” 还是 “**机构D：4**” 了！
底层挂载了强悍的双向捕获正则表达式，支持全站任意混排格式的数字与中文/英文字母组合解析。同时兼容实务中最为棘手的 **全角括号 `（1）` 与半角括号 `(1)` 混杂排版** 的痛点，一旦匹配连根拔起全部识别！

### 3. 🚀 浏览器本地加密级缓存 (LocalStorage)
每次刷新网页或者隔天打开都需要重新填写厚厚的标记词典？系统默认集成您的浏览器缓存机制：你在“标记说明”字典中输入的每一个字，都会被防丢进程即刻快照保存。甚至还配备了优雅的“清空缓存”防呆预警按钮。

### 4. 💎 殿堂级 Glassmorphism 现代极客面板
打破枯燥陈旧的基础工具界面！
深色模式下的极客向半透明玻璃态材质视觉设计，搭配**悬浮 Toast 动态滑动响应提示框**（告别原生 `alert()` 难看阻断的弹窗流程），并且内建了一键顺滑复制到系统剪贴板的辅助操作区设定。

## 🛠️ 安装与运行指南

本工具为纯前端轻量化页面架构，部署与运行门槛极低 (Zero Dependencies)。

1. **直接启动**：双击根目录下的 `标记助手.html` （或旧版的 `冬冬七号.html`）直接在现代浏览器中打开即可开始服役。
2. **离线使用**：整个系统为客户端封闭渲染闭环，无需任何网络接口支援，核心词典存储于浏览器的 LocalStorage 中，最大限度保证保密类文本不在外网中途泄露。

---

## 💡 最佳使用流示范

1. **设定标记库（自动保存）：** 
   在顶部的配置框随意粘贴或输入图件名称，例如：`1 面皮机构; 2 传送带; 注馅机构-3`。
2. **粘贴原文：**
   将需要排版的生草专利文本粘贴到左侧的【原文工作区】。
3. **一键换装：**
   点击中间绚丽的动作按钮块（例如：**发明内容 ⇒ 实施例**）。
4. **一键提单：**
   右侧的【编译输出区】即刻生成排版精良的标准专利段落文本，点击右上角的 **✨ 一键复制** 无缝衔接投入Word文件。
